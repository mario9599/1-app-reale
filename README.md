## Funzionalità

- Ricerca meteo per città
- Temperatura in tempo reale
- Umidità
- Gestione errori

## Tecnologie

- React Native
- Expo
- TypeScript
- OpenWeatherMap API

## Installazione

```bash
git clone https://github.com/mario9599/1-app-reale.git
cd 1-app-reale
npm install
npx expo start
```

## Struttura progetto

```
app/          → schermate
components/   → componenti riutilizzabili
services/     → logica API
hooks/        → hook personalizzati
constants/    → colori e configurazione
types/        → tipi TypeScript
```
